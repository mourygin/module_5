Пользователи:

Admin (psw - Admin) - пользователь с администраторскими полномочиями (Создание, удаление и редактирование новых пользователей и новых фильмов. Загрузка списков пользователей и фильмов)

User-max (psw - User-max) - рядовой пользователь.

User-min (psw - User-min) - несовершеннолетний пользователь. В его списке фильмов не отображаются фильмы, имеющие возрастные ограничения.

Bad user (psw - bad) - пользователь, аккаунт которого заблокирован.


Файлы module5hard_Films.csv и module5hard_Users.csv содержат предназначенные для загрузки списки пользователей и фильмов, полученные из неких внешних источников.
